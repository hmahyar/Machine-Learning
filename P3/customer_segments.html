Choose a Cluster Type¶

5) What are the advantages of using K Means clustering or Gaussian Mixture Models?

Answer: These two Models act really like each other. I think only differences between these two models can be about type of prediction. K-Means predicts deterministic (hard) so it is faster, especially when we have small numbers of K. Time complexity for K-Means is normally O (n^2 log n), Whereas Gaussian Mixture predicts more probabilistic (Soft). So it makes less mistake in comparison with k-Means in cases data points are near decision boundary.

I want to use K-Means because it is deterministic and we dont need to deal with probability at this point. Because we dont know after how much certainty we can assign a data point to a cluster.
